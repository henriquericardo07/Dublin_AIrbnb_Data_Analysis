# Dublin_AIrbnb_Data_Analysis
Dublin Airbnb Data Analysis: The goal is to explore how different factors such as location, property type and availability influence Airbnb listings in the housing problem that the city has.


## Project Objective

The objective of this project is to perform Exploratory Data Analysis (EDA) on Airbnb data to identify trends and insights related to:

- Listing prices
- Property types
- Availability
- Review activity
- Location-based differences

This type of analysis can help understand short-term rental markets and tourism patterns.

---

## Dataset

The dataset used for this project was obtained from Inside Airbnb, but chose especificaly Ireland.

Dataset includes information such as:

- Listing ID
- Price
- Property type
- Room type
- Number of reviews
- Availability
- Host information
- Location

Source:
https://insideairbnb.com

## 🔎 Analysis Performed

### Price Distribution
Analysis of listing prices across different properties.

### Property Types
Identification of the most common property types.

### Availability
Evaluation of property availability throughout the year.

### Reviews Activity
Analysis of the number of reviews and their distribution across listings.

### Location Insights
Understanding how location influences listing prices and demand.
